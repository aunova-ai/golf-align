# GolfAlign 수정 및 테스트 로그 - 2026-06-20

## 반영한 수정

1. 배포 환경에서 Google Sheets가 연결되지 않았을 때 Vercel 서버 파일 DB에 쓰지 않도록 수정했다.
   - 운영 환경에서는 `GOOGLE_SHEETS_NOT_CONFIGURED` 응답을 돌려주고, 화면은 해당 기기 저장 모드로 계속 사용할 수 있게 한다.
   - 회원가입 후 같은 기기에서 다시 로그인하는 프로토타입 흐름이 끊기지 않도록 했다.

2. 마지막 로그인 아이디 기억 방식을 확인했다.
   - 아이디만 `localStorage`에 저장한다.
   - 비밀번호는 저장하지 않는다.
   - 다른 사용자 계정 목록은 화면에 노출하지 않는다.

3. 훈련 체크 상태 저장 흐름을 보완했다.
   - 회원이 훈련을 누르면 화면에서 즉시 `완료` / `진행 중`으로 바뀐다.
   - `/api/training-assignments` `PATCH` API를 추가해 `training_assignments.status`를 갱신한다.
   - 로컬 프로토타입 DB와 Google Sheets 연결 상태를 모두 고려했다.

4. 영상/이미지 저장 정책은 기존 MVP 방향을 유지한다.
   - 프로필 사진: Google Drive 저장, DB에는 경로 저장.
   - 훈련 영상: 원본 장기 보관은 사용자 기기 또는 NAS 중심.
   - 서버/DB에는 썸네일, 스냅샷, 피드백 메타데이터 중심으로 저장.
   - 영상은 피드백 완료 후 7일 또는 업로드 후 14일 기준으로 NAS 이동 후보가 된다.

## 직접 확인한 테스트

1. `npm.cmd run build`
   - 통과.

2. 로컬 서버
   - `http://localhost:3000` 응답 200 확인.

3. API 저장 흐름
   - 테스트 회원가입 성공.
   - 테스트 회원 로그인 성공.
   - 훈련 과제 생성 성공.
   - 훈련 과제 상태 `완료` PATCH 성공.
   - GET 재조회 시 같은 과제의 상태가 `완료`로 유지됨.

4. 화면 테스트
   - 로그인 화면 한글 렌더링 정상.
   - 관리자 계정 로그인 화면 진입 정상.
   - 회원 화면의 훈련 탭 진입 정상.
   - 훈련 체크 클릭 시 `aria-pressed=true`, 화면 텍스트 `완료`로 변경.
   - 브라우저 콘솔 오류 없음.
   - 로그아웃 후 마지막 로그인 아이디만 남고 비밀번호는 비어 있음.
