# GolfAlign MVP 화면 구조 v0.4

작성일: 2026-05-09  
상태: 커리큘럼 확정 전 기준안  
목적: 실제 개발 전 회원 화면, 프로 화면, 초대/가입, 업로드, 훈련 기록 흐름을 하나의 기준으로 정리한다.

## 1. 기본 방향

GolfAlign MVP는 웹앱/PWA 기준으로 먼저 만든다.  
정식 앱 출시 전까지는 브라우저 접속, 홈 화면 추가, Vercel 배포, 보유 도메인 연결을 기준으로 테스트한다.

MVP의 핵심은 다음 5가지다.

1. 회원이 스윙/퍼팅/어프로치 영상 또는 이미지를 기록한다.
2. 회원은 기록을 개인 보관하거나, 선택적으로 프로방에 공유한다.
3. 프로는 회원 기록을 보고 짧은 피드백과 스냅샷 메모를 남긴다.
4. 프로는 회원에게 훈련 과제를 보내고, 회원은 결과를 체크리스트처럼 기록한다.
5. 프로방 초대 링크를 통해 가입률과 유지율을 높인다.

커리큘럼 추천 목록은 프로와 협의 후 반영한다.  
v0.4에서는 추천 과정 내용을 고정하지 않고, 구조만 받을 수 있게 비워둔다.

## 2. 사용자 역할

### 일반 회원

일반 회원은 앱의 기본 기능을 혼자 사용할 수 있다.

- 내 기록 업로드
- 기록 상세 확인
- 영상/이미지 기반 피드백 확인
- 개인 훈련 기록
- 프로방 가입
- 프로방에 선택 기록 공유
- 받은 훈련 과제 수행
- 언어 설정

회원은 프로방에 들어가지 않아도 자신의 기록과 일반 기능을 사용할 수 있다.  
프로방 가입 후에는 기존 기록 중 일부를 선택해서 프로에게 공유할 수 있다.

### 프로

프로는 회원을 1:1 중심으로 관리한다.

- 프로방 생성 및 관리
- 초대 링크 생성
- 가입 회원 목록 확인
- 회원별 기록 확인
- 스냅샷 피드백 작성
- 훈련 과제 전송
- 훈련 결과 확인
- 짧은 코멘트 작성

프로방 내 회원 목록은 이름과 프로필 이미지만 공개한다.  
다른 회원의 세부 기록, 영상, 피드백, 훈련 결과는 공개하지 않는다.

### 센터/관리자

MVP에서는 화면을 만들지 않는다.  
추후 확장 단계에서 프로 인력관리, 회원 현황, 매출 계산, 센터별 통계를 추가한다.

## 3. 회원 화면 구조

### 3.1 홈

목적: 오늘 해야 할 일과 최근 기록을 빠르게 보여준다.

구성:

- 상단 로고
- 회원 홈 요약
- 영상/이미지 업로드 버튼
- 새 피드백 알림
- 오늘 할 훈련
- 최근 기록
- 라운드 모집 개발 예정 카드

주요 버튼:

- 영상/이미지 업로드
- 오늘 할 훈련 전체 보기
- 최근 기록 전체 보기
- 프로방 이동

### 3.2 내 기록

목적: 회원이 업로드한 스윙, 퍼팅, 어프로치 기록을 모아본다.

구성:

- 새 기록 업로드 버튼
- 기록 목록
- 영상/이미지 구분
- 프로방 공유 여부 배지
- 클럽/촬영각도/기록일 정보

기록 상태:

- 개인 보관
- 프로방 공유됨
- 피드백 대기
- 피드백 완료

### 3.3 기록 업로드

목적: 영상 또는 이미지를 간단히 업로드한다.

입력 항목:

- 기록 종류: 스윙, 퍼팅, 어프로치
- 파일 종류: 영상, 이미지
- 클럽 종류
- 촬영 각도: 정면, 측면, 후방, 퍼팅 라인
- 메모
- 프로방 공유 여부

영상 정책:

- 5초 권장
- 최대 7초 허용
- 7초 이하 영상은 바로 업로드한다.
- 7초 초과 영상은 시작 지점을 선택하고, 선택 지점부터 최대 7초만 저장한다.
- 업로드 직후 시작 자세, 중간 자세, 마무리 자세 스냅샷 3장을 만든다.
- 앱 화면은 스냅샷과 썸네일을 먼저 보여준다.
- 원본 영상은 앱에서 스트리밍하지 않고, 필요할 때 다운로드 후 기기에서 재생한다.

### 3.4 기록 상세

목적: 영상/이미지 중심으로 분석과 피드백을 확인한다.

구성:

- 스냅샷/이미지 뷰어
- 시작 자세, 중간 자세, 마무리 자세 탭
- 본/각도/라인 표시
- 스티커 메모
- 간단 피드백
- 원본 영상 다운로드 버튼
- 프로방 공유 버튼
- Before & After 비교 진입

스티커 메모 원칙:

- 영상 정지 스냅샷 기준으로 작성
- 영상 전체 재생 없이도 스냅샷 위에 작성 가능해야 한다.
- 기본은 짧은 말풍선
- 누르면 상세 코멘트 확인
- 화면을 많이 가리지 않게 배치

### 3.5 훈련

목적: 프로가 보낸 과제와 개인 훈련을 투두리스트처럼 관리한다.

구성:

- 전체
- 프로 추천
- 개인 훈련
- 완료
- 접었다 펼칠 수 있는 훈련 카드

훈련 카드 기본 정보:

- 훈련명
- 출처: 프로방 공통 드릴, 개인 과제, 개인 훈련
- 목표
- 반복 수 또는 시간
- 마감일
- 영상/이미지 첨부 필요 여부

### 3.6 훈련 결과 기록

목적: 훈련 수행 결과를 간단히 남긴다.

입력 항목:

- 완료 여부
- 실제 반복 수
- 성공 수
- 난이도: 쉬움, 보통, 어려움
- 짧은 메모
- 영상/이미지 선택 업로드

미디어 업로드는 선택으로 둔다.  
훈련에 따라 프로가 필수로 지정할 수 있는 구조는 나중에 추가한다.

### 3.7 프로방

목적: 회원이 가입한 프로방과 받은 과제를 확인한다.

구성:

- 가입한 프로방 목록
- 프로 정보 한 줄 요약
- 최근 피드백
- 공통 드릴
- 개인 과제
- 초대 코드 입력

프로 정보는 길게 노출하지 않는다.  
상세 정보는 별도 화면에서 확인하게 한다.

### 3.8 초대/가입 흐름

목적: 카카오톡, 문자, 메시지에 공유된 링크로 쉽게 들어오게 한다.

흐름:

1. 프로가 초대 링크 생성
2. 카카오톡/문자/메시지로 링크 공유
3. 사용자가 링크 클릭
4. 가입된 회원이면 프로방 참여 화면으로 이동
5. 미가입 회원이면 가입 화면으로 이동
6. 가입 완료 후 초대된 프로방 확인
7. 기존 기록을 공유할지 선택

중요 원칙:

- 초대 링크는 가입 유도와 방 참여를 동시에 처리한다.
- 가입하지 않아도 초대된 방 정보는 간단히 보여준다.
- 가입 후 기존 개인 기록을 자동 공유하지 않는다.
- 회원이 선택한 기록만 프로방에 공유한다.

### 3.9 더보기/설정

목적: 계정, 언어, 앱 정보, 저장 정책을 관리한다.

구성:

- 내 프로필
- 언어 설정
- 알림 설정
- 저장 공간 안내
- 앱 정보
- 문의

언어 설정:

- 한국어
- 영어

기본 언어 결정:

1. 사용자가 선택한 언어
2. 계정에 저장된 선호 언어
3. 브라우저/기기 언어
4. 서비스 기본값

한국 MVP는 기본값을 한국어로 둔다.  
글로벌 배포 시 앱스토어/플레이스토어 지역과 언어 정책을 다시 검토한다.

## 4. 프로 화면 구조

### 4.1 프로 홈

목적: 오늘 처리해야 할 회원 관리 작업을 빠르게 보여준다.

구성:

- 프로 로고/배지
- 피드백 대기 수
- 훈련 결과 확인 수
- 신규 가입 회원
- 최근 요청
- 빠른 작업 버튼

빠른 작업:

- 회원방 보기
- 초대 링크 만들기
- 피드백 작성
- 훈련 결과 확인

### 4.2 프로방 목록

목적: 프로가 운영 중인 방을 관리한다.

구성:

- 방 이름
- 가입 회원 수
- 피드백 대기 수
- 최근 활동
- 초대 링크 버튼

프로방은 1:1 관리 중심으로 운영한다.  
방 안에서 공통 드릴은 공개 가능하지만, 개인 결과는 비공개다.

### 4.3 프로방 상세

목적: 특정 프로방의 회원과 과제를 관리한다.

구성:

- 방 정보
- 가입 회원 목록
- 공통 드릴
- 개인 과제
- 초대 링크

회원 목록 공개 범위:

- 이름
- 프로필 이미지
- 가입 상태

비공개 범위:

- 개인 기록
- 영상/이미지
- 피드백
- 훈련 결과
- 개인 메모

### 4.4 회원 상세

목적: 프로가 한 회원의 기록과 훈련 상태를 본다.

구성:

- 회원 기본 정보
- 최근 업로드 기록
- 공유된 기록
- 받은 피드백
- 진행 중 훈련
- 완료한 훈련 결과

프로 액션:

- 기록 보기
- 스냅샷 피드백 작성
- 훈련 과제 보내기
- 짧은 코멘트 남기기

### 4.5 피드백 작성

목적: 영상/이미지 스냅샷 위에 간단한 피드백을 남긴다.

구성:

- 영상/이미지 뷰어
- 정지 스냅샷 선택
- 스티커 메모
- 목표 코멘트
- 신경쓸 점
- 다음 과제 추천

피드백 문장 구조:

- 목표: 이번 주에는 어드레스 정렬을 먼저 맞춥니다.
- 신경쓸 것: 백스윙 때 왼쪽 어깨가 너무 빨리 열리지 않게 해주세요.
- 다음 과제: 하프스윙 50개 후 측면 영상 1개를 올려주세요.

### 4.6 훈련 결과 확인

목적: 회원이 기록한 훈련 결과를 보고 짧은 코멘트를 남긴다.

구성:

- 과제명
- 회원 결과
- 반복 수
- 성공 수
- 난이도
- 첨부 영상/이미지
- 프로 코멘트

예시 코멘트:

어프로치는 훈련기록과 영상으로 충분히 좋아졌습니다.  
다음 주에는 퍼팅 거리감 과제로 넘어가겠습니다.

## 5. 라운드 모집 기능

MVP에서는 기능을 비활성화한다.  
다만 추후 확장을 위해 카드 형태만 노출한다.

현재 노출:

- 라운드 모집 카드
- 개발 중 배지
- 참여자 프로필 이미지 슬롯

추후 확장:

- 모집글 작성
- 참여 신청
- 참가자 프로필
- 일정/장소
- 채팅 또는 알림

## 6. 하단 내비게이션

### 회원

- 홈
- 기록
- 훈련
- 프로방
- 더보기

### 프로

- 홈
- 회원방
- 피드백
- 훈련확인
- 더보기

실제 앱에서는 회원/프로 전환 버튼을 노출하지 않는다.  
로그인한 계정의 역할에 따라 자동으로 화면을 분기한다.

## 7. MVP에서 보류할 것

다음 기능은 구조만 열어두고, MVP에서 무리하게 구현하지 않는다.

- AI 정밀 분석 고도화
- 3D 모델/리깅 모델
- 앱스토어/플레이스토어 출시
- 결제
- 센터 관리자 화면
- 라운드 모집 실제 기능
- 전체 커뮤니티
- 복잡한 소셜 기능
- 커리큘럼 자동 추천
